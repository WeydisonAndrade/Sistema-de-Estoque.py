# Sistema-de-Estoque.py
Sistema de Controle de Estoque

Projeto acadêmico desenvolvido em Python utilizando SQLite.

Funcionalidades:
- Login de usuários autorizados
- Cadastro e exclusão de usuários
- Entrada e saída de produtos
- Controle de estoque
- Registro de movimentações com data e hora

Tecnologias:
- Python
- SQLite

O que aprendi:
- Modelagem de dados
- Lógica de sistemas
- Regras de negócio
